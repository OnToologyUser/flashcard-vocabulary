Cardc back to ToC or Class ToC

IRI: https://w3id.org/flashcards/vocab/2018#Card

A physical or virtual card bearing information such as words, images or numbers, on either or both sides. Used in classroom drills or in private study. The card has an associated test and an answer.

Stackc back to ToC or Class ToC

IRI: https://w3id.org/flashcards/vocab/2018#Stack

A stack represents a collection of cards and is typically used to group cards belonging to a topic being studied. E.g. a collection of cards containing vocabulary for a chapter in a language textbook.
is in domain of
instruction dp, media dp, media url dp, related work op
is disjoint with

sort orderdp back to ToC or Data Property ToC

IRI: https://w3id.org/flashcards/vocab/2018#sortOrder

The initial card sort order expressed as an integer value. The sort order should be used as a suggestion, e.g. when no other sort order exists, and may be ignored by users and tools. The sort order does not imply that consecutive cards have a sort order incrremented by one.
